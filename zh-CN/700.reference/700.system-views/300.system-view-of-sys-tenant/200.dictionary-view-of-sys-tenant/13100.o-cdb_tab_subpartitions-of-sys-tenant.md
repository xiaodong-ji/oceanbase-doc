# oceanbase.CDB_TAB_SUBPARTITIONS

## 功能

展示所有租户的分区表的二级分区信息。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始引入。</p>
</main>
