# DBA_TAB_PARTITIONS

## 功能

展示当前用户可访问的多级分区信息、分区存储参数和分区统计信息。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V2.2.77 版本开始引入。</p>
</main>

## 相关视图

* `ALL_TAB_PARTITIONS`

* `USER_TAB_PARTITIONS`
