# DBA_TAB_SUBPARTITIONS

## 功能

展示子分区名称、表和所属分区的名称、其存储属性以及 `DBMS_STATS` 包生成的统计信息。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V2.2.77 版本开始引入。</p>
</main>

## 相关视图

* `ALL_TAB_SUBPARTITIONS`

* `USER_TAB_SUBPARTITIONS`
