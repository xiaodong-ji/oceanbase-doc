ALL_PART_TABLES
====================================

功能
-----------

展示当前用户可访问的分区表的分区信息。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库企业版。OceanBase 数据库社区版仅提供 MySQL 模式。</p>
  </main>

相关视图
-------------

* DBA_PART_TABLES

* USER_PART_TABLES
