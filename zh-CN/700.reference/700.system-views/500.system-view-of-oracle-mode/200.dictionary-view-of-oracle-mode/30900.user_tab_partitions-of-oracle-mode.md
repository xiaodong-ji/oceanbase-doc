USER_TAB_PARTITIONS
========================================

功能
-----------

查看属于当前用户的多级分区信息、分区存储参数和分区统计信息。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库企业版。OceanBase 数据库社区版仅提供 MySQL 模式。</p>
  </main>
