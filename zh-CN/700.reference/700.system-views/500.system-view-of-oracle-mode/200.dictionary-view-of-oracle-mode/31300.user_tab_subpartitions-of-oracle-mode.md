# USER_TAB_SUBPARTITIONS

功能
-----------

描述了当前用户拥有的每个表子分区的子分区名、表的名称和它所属的分区以及它的存储属性。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库企业版。OceanBase 数据库社区版仅提供 MySQL 模式。</p>
  </main>
