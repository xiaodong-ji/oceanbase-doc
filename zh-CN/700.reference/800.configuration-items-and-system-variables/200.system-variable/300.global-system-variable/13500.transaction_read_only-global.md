# transaction_read_only

`transaction_read_only` 用于控制是否仅允许开启只读事务。

|   属性    |                                                     描述                                                     |
|---------|------------------------------------------------------------------------------------------------------------|
| 参数类型    | bool                    |
| 默认值     | False                   |
| 取值范围    | <li> True   <li> False        |
| 生效范围    | <li> Global   <li> Session    |
| 是否可修改  | 是，支持通过 `SET` 语句修改。|

当 `transaction_read_only` 设置为 `True` 时，不允许执行 DML 语句。
