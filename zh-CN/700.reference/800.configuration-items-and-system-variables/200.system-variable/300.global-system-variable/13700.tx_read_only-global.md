# tx_read_only

tx_read_only 用于设置是否为只读事务。

| **属性**  |                                                   **描述**                                                    |
|---------|-------------------------------------------------------------------------------------------------------------|
| 参数类型    | int                      |
| 默认值     | 0                        |
| 取值范围    | <li> 1：只读事务   <li> 0：不是只读事务    |
| 生效范围    | <li> GLOBAL   <li> SESSION     |
| 是否可修改  | 是，支持通过 `SET` 语句修改。|

  <main id="notice" type='explain'>
    <h4>说明</h4>
    <p>设置后，仅影响该用户，不影响其他用户。</p>
  </main>
