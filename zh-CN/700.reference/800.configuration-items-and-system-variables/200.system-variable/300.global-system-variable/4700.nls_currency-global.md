# nls_currency

`nls_currency` 用于设置数字格式元素 `L` 的本地货币符号。

  <main id="notice" type='explain'>
    <h4>说明</h4>
    <p>该变量不支持修改。</p>
  </main>

|   属性    |                                                     描述                                                     |
|---------|------------------------------------------------------------------------------------------------------------|
| 参数类型    | varchar                 |
| 默认值     | $                       |
| 取值范围    | N/A                     |
| 生效范围    | <li> Global   <li> Session    |
| 是否可修改    | 否  |
