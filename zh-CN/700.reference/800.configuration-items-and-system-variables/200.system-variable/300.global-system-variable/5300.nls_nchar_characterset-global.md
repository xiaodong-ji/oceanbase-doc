# nls_nchar_characterset

`nls_nchar_characterset` 表示数据库默认字符集，用于 NCHAR、NVARCHAR2、NCLOB 等数据类型。

  <main id="notice" type='explain'>
    <h4>说明</h4>
    <p>该变量不支持修改。</p>
  </main>

|  **属性**  |  **描述**   |
|----------|-----------|
| 参数类型     | varchar   |
| 默认值      | AL16UTF16 |
| 取值范围     | N/A       |
| 生效范围     | Global    |
| 是否可修改    | 否  |
