# ob_bnl_join_cache_size

ob_bnl_join_cache_size 用于设置 batch nest loop join 一次 cache 多少数据做一次 batch。

|  **属性**  |                                                   **描述**                                                   |
|----------|------------------------------------------------------------------------------------------------------------|
| 参数类型     | int                     |
| 默认值      | 10485760                |
| 取值范围     | \[1, 9223372036854775807\]                                                                                 |
| 生效范围     | <li> GLOBAL   <li> SESSION    |
| 是否可修改 | 是，支持通过 `SET` 语句修改。|
