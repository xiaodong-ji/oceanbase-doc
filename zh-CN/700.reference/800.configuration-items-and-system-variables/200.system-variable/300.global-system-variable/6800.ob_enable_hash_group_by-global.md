## ob_enable_hash_group_by 


<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始弃用。</p>
</main>

`ob_enable_hash_group_by` 用于设置是否打开 Hash Group by 的路径。

|  **属性**  |                                                   **描述**                                                   |
|----------|------------------------------------------------------------------------------------------------------------|
| 参数类型     | bool                                                                                                       |
| 默认值      | 1                                                                                                          |
| 取值范围     | </li><li> 0：不允许   </li><li> 1：允许        |
| 生效范围     | </li><li> GLOBAL   </li><li> SESSION    |
