# ob_temp_tablespace_size_percentage

<main id="notice" type='explain'>
<h4>说明</h4>
<p>该变量已不再使用，临时空间由 data_disk_usage_limit_percentage 配置项控制。</p>
</main>

ob_temp_tablespace_size_percentage 用于租户磁盘中临时表空间大小的百分比限制。


| **属性** |                                                   **描述**                                                   |
|--------|------------------------------------------------------------------------------------------------------------|
| 参数类型   | int                     |
| 默认值    | 0                       |
| 取值范围   | \[0, 100\]              |
| 生效范围   | <li> GLOBAL   <li> SESSION    |
